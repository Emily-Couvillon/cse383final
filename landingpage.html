About This Project

This page uses a navbar to allow a user to navigate the website. This navbar is available on all three pages and collapses when the viewport is shrunk. The stock page allows a user to select a stock from a dropdown. Once this is done: the page will display the stock and a user can then find further information via a 'Details' button or a 'News' button. The third page allows a user to check the history of stock searches and then displays these results in a neat table.
